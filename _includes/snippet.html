{%- comment -%} Collapsible, Slack-style code snippet. Paste the raw file into _includes/snippets/ then reference it: {% include snippet.html file="hotel-trip-planner-spec.md" lang="markdown" %} Params: file (required) — filename inside _includes/snippets/ lang (optional) — highlighter language (bash, python, ruby, js, java, yaml, …) lines (optional) — lines shown while collapsed (default: 5) title (optional) — label in the header bar (defaults to the filename) Styles + behaviour are self-contained here (no dependency on style.css), so the collapse works even if the SCSS hasn't been recompiled/deployed. The critical clamp uses literal units, so it holds even if the design tokens are absent. Note: the snippet file is run through Liquid.
